# Neural Network Analysis Project

## Objective
This project demonstrates neural network model building, training, evaluation, and hyperparameter experimentation using Python and TensorFlow/Keras.

## Tasks Completed
- Dataset Understanding
- Data Preprocessing
- Neural Network Model Building
- Training and Evaluation
- Hyperparameter Experimentation
- Final Reflection

## Technologies Used
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Files Included

### notebook.ipynb
Contains complete implementation of all tasks.

### requirements.txt
Contains required Python libraries.

### results/model_comparison_table.csv
Comparison table of hyperparameter experiments.

### results/evaluation_outputs.png
Contains evaluation graphs and confusion matrix.

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Open Jupyter Notebook:
   jupyter notebook

3. Run notebook.ipynb
